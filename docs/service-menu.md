# Web3 兼职服务菜单

这份菜单用于和项目方沟通小任务范围。实际报价和周期需要根据项目复杂度、资料完整度和修改次数确认。

## 中文内容与教程

适合任务：

- 项目中文介绍
- 测试网交互教程
- 钱包连接 / Mint / Claim 用户教程
- FAQ 和用户上手文档
- 公告、活动文案、社区帖子

可交付物：

- Markdown / Notion / Google Docs 文档
- 截图标注版教程
- 中英文双语初稿
- 发布前检查清单

## 社区运营支持

适合任务：

- Discord / Telegram 内容日历
- 活动规则整理
- 用户问题分类
- 每周社区摘要
- Ambassador / Quest 活动文案

可交付物：

- 周报模板
- 活动文案
- FAQ 整理表
- 社区内容排期

## dApp 前端 Demo

适合任务：

- 钱包连接页面
- NFT Mint 页面
- Claim / Faucet / Allowlist 静态原型
- 交易状态 UI
- 测试网交互流程页

可交付物：

- 响应式 HTML / CSS / JS 页面
- Vite / Next.js 前端原型
- 静态部署链接
- README 和使用说明

## Web3 项目分析

适合任务：

- 项目定位分析
- 用户流程拆解
- 竞品页面分析
- 风险边界整理
- 中文社区介绍材料

可交付物：

- 项目分析文档
- 用户流程说明
- 风险检查清单
- 面向社区的简明介绍

## 合作建议

- 先从一个小任务开始。
- 每个任务先确认交付范围、参考资料、截止时间和修改次数。
- 不接触私钥、助记词、代操作账户、押金或异常资金流转。
- 涉及真实链上交易的任务需要额外确认测试网、合约地址和风险提示。
