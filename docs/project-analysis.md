# Web3 项目分析样例

## 项目定位

Proof Credential 是一个轻量级 NFT 凭证 demo，用于展示 Web3 兼职交付能力。它不是主网金融产品，而是一个适合测试网、活动凭证、社区身份和教程场景的原型。

## 用户场景

- 活动参与者领取链上凭证
- 社区成员获得入群或白名单证明
- 教程用户完成任务后领取 NFT
- 项目方用 demo 快速展示 Web3 onboarding 流程

## 核心流程

1. 用户连接钱包
2. 页面读取钱包状态
3. 用户点击 mint
4. 前端展示交易哈希和 mint 结果
5. NFT 可作为凭证在个人资料或社区中展示

## 风险检查

- 每个地址限制 mint 1 个，避免重复领取
- 设置最大供应量，避免无限铸造
- 管理员只允许更新 metadata base URI
- 免费 mint 不涉及收款和提现，降低资金风险

## 可升级方向

- 接入真实测试网钱包连接
- 增加白名单签名 mint
- 使用 IPFS / Arweave 保存 metadata
- 增加链上任务验证
- 增加中文教程和视频演示
